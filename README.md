# Healer's Odyssey

A lifelong medical study RPG. Level up through 38 specialties, 1,784+ diseases, USMLE-style AI quizzes, spaced repetition, and more.

**Live site:** https://YOUR-USERNAME.github.io/healers-odyssey/

## Features
- 38 medical specialties unlocking from Level 1 → Level 102
- USMLE Step 1/2 AI-generated clinical vignettes
- Spaced repetition with confidence ratings
- Activity heatmap, focus timer, hot streak combo
- Cloud sync via JSONBin.io
- Flashcard mode, study planner, custom topics
- Prestige system (up to 5×, +75% XP)
- 10-track RPG lofi playlist

## Deploy
Push `index.html` to a GitHub repo, enable GitHub Pages under Settings → Pages.
